# library back-end

Projeto de avaliação para processo seletivo do INDT.

## Start 

1. Instalar Depedências:
   
   ```sh
   npm install
   ```

2. Executar o Projeto:
   
   ```sh
   npm run dev
   ```
   
## Depedencias do Projeto 

    -- express
    -- http-status
    -- mongoose

## Build 
   
   ```sh
   npm run build
